# Package index

## Start Up

- [`chattr_use()`](https://mlverse.github.io/chattr/reference/chattr_use.md)
  : Sets the LLM model to use in your session

## Main functions

- [`chattr_app()`](https://mlverse.github.io/chattr/reference/chattr_app.md)
  : Starts a Shiny app interface to the LLM
- [`chattr()`](https://mlverse.github.io/chattr/reference/chattr.md) :
  Submits prompt to LLM

## Session defaults

- [`chattr_defaults()`](https://mlverse.github.io/chattr/reference/chattr_defaults.md)
  : Default arguments to use when making requests to the LLM
- [`chattr_defaults_save()`](https://mlverse.github.io/chattr/reference/chattr_defaults_save.md)
  : Saves the current defaults in a yaml file that is compatible with
  the config package

## Utilities

- [`ch_history()`](https://mlverse.github.io/chattr/reference/ch_history.md)
  : Displays and sets the current session' chat history
- [`chattr_test()`](https://mlverse.github.io/chattr/reference/chattr_test.md)
  [`ch_test()`](https://mlverse.github.io/chattr/reference/chattr_test.md)
  : Confirms connectivity to LLM interface
